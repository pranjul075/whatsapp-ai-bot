# 🤖 AI WhatsApp Bot

An AI-powered WhatsApp bot that automatically replies to messages using AI.

## 🚀 Features

- WhatsApp automation
- AI generated responses
- Real-time message handling

## 🛠 Tech Stack

- Node.js
- whatsapp-web.js
- Google Gemini API

## 📦 Installation

Clone the repository

git clone https://github.com/pranjul075/whatsapp-ai-bot.git

Go to project folder

cd whatsapp-ai-bot

Install dependencies

npm install

Run the bot

node bot.js

Scan the QR code using WhatsApp.
